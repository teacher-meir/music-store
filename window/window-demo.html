BOM - Browser Object Model

The Browser Object Model (BOM) provides an API for interacting with the browser window and its components.

Window Object

The window object represents the browser window and provides properties and methods for controlling the window's behavior and appearance.

Properties

  • window.innerWidth: Returns the width of the window's content area.
  • window.innerHeight: Returns the height of the window's content area.
  • window.location: Provides information about the current URL and allows you to navigate to different URLs.
  • window.navigator: Provides information about the browser and its capabilities.
  • window.screen: Provides information about the user's screen.

Methods

  • window.print(): Opens the print dialog to print the current page.
  • window.open(): Opens a new browser window or tab.
  • window.close(): Closes the current browser window.
